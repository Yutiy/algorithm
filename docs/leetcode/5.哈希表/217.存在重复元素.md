# 题目地址

https://leetcode-cn.com/problems/contains-duplicate/

## 解题思路

采用 Set 数据结构去重，然后与原先数组长度进行对比

## 解题代码

```js
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function(nums) {
  // return Array.from(new Set(nums)).length != nums.length;
  return new Set(nums).size != nums.length;
};
```
